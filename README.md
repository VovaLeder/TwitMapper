### Установка
* Должны быть установлен `npm`
* Клонируйте репу
* Заходите через терминал в `/twitmapper`
* Пропишите: `npm install`

### Запуск
* Заходите через терминал в `/twitmapper`
* Пропишите: `npm start`

Без интернета и запущена бека Демида дальше страницы регистрации уйти не сможете
